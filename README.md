# PentaHospitals - Backend Architecture Showcase

> **Disclaimer:** The source code for this project is strictly private due to a Non-Disclosure Agreement (NDA). This repository serves solely as a showcase of my technical responsibilities, architecture design, and the technologies used during the project.

##  Project Overview
An automated notification system developed for PentaHospitals to inform patients about the availability of their medical test results. The system is fully integrated with the facility's existing external patient database.

##  My Role: Backend Developer
I am working as part of a 4-person development team. My main responsibilities focus entirely on the backend infrastructure:
* Designing and creating the foundational project architecture (skeleton).
* Developing the core business logic for the patient notification flow.
* Integrating the new system with the company's external database.
* Ensuring secure, stable, and smooth data transfer between the database and the application.
